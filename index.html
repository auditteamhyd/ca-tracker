🐙 GitHub Gist Setup

🔑 Step 1 — Create a Personal Access Token (PAT)
  1. Go to github.com → click your avatar → Settings
  2. Scroll to Developer settings (bottom-left) → Personal access tokens → Tokens (classic)
  3. Click Generate new token (classic) → give it a name like CA Tracker
  4. Expiration: No expiration (or 1 year) · Scope: tick gist only
  5. Click Generate token → copy the token starting with ghp_
📁 Step 2 — Data storage
On first Push to Gist, the app automatically creates a secret Gist named ca_tracker_data.json under your GitHub account.
You can also paste an existing Gist ID to reuse it across devices.
Secret Gists are not indexed by search engines but are accessible to anyone with the direct URL — your data is app-only (no sensitive PII like passwords is stored here; PINs stay in localStorage).
🚀 Step 3 — Host on GitHub Pages
  1. Create a GitHub repo → upload this file as index.html
  2. Settings → Pages → Source: main branch / root → Save
  3. Your app URL: https://<yourusername>.github.io/<repo-name>/
  4. Open that URL → enter your PAT in the setup below → done ✅
